# 02 — Product Design
Apply the design principles (`00-governance/PRINCIPLES.md`). Review checklist:
- Visual hierarchy: the one key fact per view is obvious at a glance.
- All states designed: empty / loading / error-offline / success.
- Accessibility: contrast, keyboard, focus, ARIA. RTL + dark mode verified.
- Microcopy in EN + AR. Icons are inline SVG — never emojis as icons.
- Progressive disclosure: detail on demand, calm by default.
Benchmark the design against world-class peers (`16-best-practices/BENCHMARK-ENGINE.md`).
