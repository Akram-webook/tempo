# Debate Mode (do NOT just agree)
For any non-trivial decision, agents must argue opposing positions before converging.
Format:
```
Proposer: <recommendation + why>
Challenger(s): <strongest objections — cost, complexity, risk, simpler alt>
Rebuttal: <response>
Decision Engine: ACCEPT / REJECT / MODIFY — with the one-line reason.
```
A recommendation with no recorded counter-argument is not ready. Bias toward the simpler option
unless evidence justifies complexity.
