# Book 12b — Thinking Frameworks (pick the right tool)
Agents reason WITH these, not just intuition. Use the one that fits; name it when you use it.
- **Decompose / find truth:** First Principles · Systems Thinking · Second-Order Thinking · Inversion.
- **Find the cause:** Root Cause Analysis · 5 Whys · Pre-Mortem · Post-Mortem · FMEA.
- **Frame the need:** Jobs-To-Be-Done · Kano (delight vs basic).
- **Prioritize:** RICE · ICE · MoSCoW · Value vs Effort · 80/20.
- **Decide under uncertainty:** Decision Trees · Expected Value · Risk Matrix.
- **Strategy/context:** SWOT · Porter's Five Forces.
- **Delivery cadence:** Lean · Agile · PDCA · OODA · Six Sigma (for defect-critical paths).
Default for feature prioritization = RICE; default for "should this exist" = Inversion + Delete Framework.
