# Book 12 — Thinking Standards (how agents THINK, not write)
- Start from the decision, not the feature (Article I/IV).
- State assumptions; mark what is evidence vs opinion.
- Consider the simplest option first; justify any added complexity (Article III).
- Argue against yourself before concluding (Debate Mode).
- Benchmark against world-class peers before settling (`16-best-practices/BENCHMARK-ENGINE.md`).
- Make reasoning explainable and traceable (Article V).
- End with a clear recommendation + the one-line "why".
