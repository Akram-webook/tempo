# 00 — Governance
How decisions get made in Tempo, and who has rights over what.

- **Decision flow:** CEO intent → Executive Strategy Partner → PM → BA → Solutions Architect →
  Senior Engineers → QA Council → Release Board. Each role reads ALL prior feedback (see `10-ai-agents`).
- **Decision rights:** Akram approves merges to `main` and any irreversible/production change.
  AI agents recommend and build; they do not merge to `main` without Akram's OK.
- **Source of truth:** the GitHub repo. Standing brief: `/CLAUDE.md` → points here.
- **Every decision is logged** in `15-knowledge-base/DECISIONS-LOG.md` (with the rejected options).

See `PRINCIPLES.md` for the principles every agent obeys.
