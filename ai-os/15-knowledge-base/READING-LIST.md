# Book 9b — Reading list (the books that build companies)
Not self-help — the canon TAOS draws from. Agents apply the ideas; humans deepen here.
- **Product:** Inspired; Empowered; Transformed (Cagan) · Continuous Discovery Habits (Torres) ·
  Escaping the Build Trap (Perri) · Lean Product Playbook (Olsen).
- **Engineering:** Clean Code; Clean Architecture (Martin) · Designing Data-Intensive Applications
  (Kleppmann) · Domain-Driven Design (Evans) · The Phoenix Project · Accelerate (Forsgren/DORA).
- **Leadership:** High Output Management (Grove) · The Manager's Path (Fournier) · Radical Candor (Scott).
- **UX:** Don't Make Me Think (Krug) · The Design of Everyday Things (Norman) · Lean UX.
- **Operations:** Toyota Production System · Lean Thinking · The Goal (Goldratt) · Measure What Matters (OKRs).
- **AI:** AI Engineering (Huyen) · Building LLM-Powered Applications.
