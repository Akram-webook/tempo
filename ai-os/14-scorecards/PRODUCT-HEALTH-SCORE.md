# Product Health Score
Score each 1–5 (or 0–100 weighted). Average → grade band. Record in the PR.
Dimensions: Business Value · Engineering Quality · Maintainability · Scalability · Security ·
Accessibility · Performance · UX · Data Quality · Operational Value · AI Readiness · Documentation ·
Testing · Automation · Adoption Risk.
Bands: 90+ Production Ready (A) · 75–89 Ship with follow-ups (B) · 60–74 Needs work (C) · <60 Block.
